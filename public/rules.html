Rules

Bid For ETH is a decentralized, trustless blockchain game running directly on the Ethereum network. Here are the basics of how BfETH works:

  • • This is a lottery game in which the last person to buy a key at the end of a round wins the pot!

  • • During a round, people can bid 1 or more time which adds time to the timer and marks them as the current leader. With each bid player receives BID token

  • • When the timer reaches zero, last bidder wins jackpot! (20% of pot goes to the next round)

  • • Before a new round, there is a 1-hour grace period in which bids are disabled and any ETH sent to the contract is routed directly to winning wallet.
